# 🏥 Healthcare Claims Voice Agent

## About the Project

The Healthcare Claims Voice Agent is a conversational AI solution built using Google CX Agent Studio. The agent automates common healthcare services such as claim status, eligibility verification, provider lookup, benefits inquiry, and pre-authorization while ensuring secure member authentication before accessing healthcare information.
